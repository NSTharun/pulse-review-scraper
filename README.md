\# Pulse Review Scraper



This project was developed as part of the Pulse Coding Assignment.



\## Objective

Scrape SaaS product reviews from G2 and Capterra within a given date range.

Bonus: TrustRadius integration.



\## Features

\- Accepts company name, start date, end date, and source

\- Outputs reviews in JSON format

\- Handles pagination

\- Graceful error handling

\- Modular and extensible design



\## Supported Sources

\- G2

\- Capterra

\- TrustRadius (Bonus)



\## Usage



```bash

python scraper.py --company hubspot --start\_date 2024-01-01 --end\_date 2024-06-30 --source g2
